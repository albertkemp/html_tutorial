认识CSS

CSS 是 html 的格式。希望您首先在样式文件夹和 html 中很好地获得了styles.css 地点。 style.css 的第一步是格式化我们将要在 html 中学习的元素。一个好方法 开始是字体。转到 Google Fonts，搜索您要使用的字体并选择使用字体或任何内容 相似的。 之后应该有两个代码，一个大的，一个小的。复制并粘贴小 <head> 元素。一个更简单的方法是简单地将下面的两个插入到 <html> 标签。

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Crimson+Text:ital@0;1&family=Libre+Baskerville&family=Stick+No+Bills:wght@200&display=swap"

然后在styles.css（您的样式表）中插入下面的代码。

html {
font-size: 18px; /* px means "pixels": the base font size is now 10 pixels high */
font-family: "Times", sans-serif; /* this should be the rest of the output you got from Google fonts */
}

2. Css Basics

首先让我们格式化一个 html 文档。复制下面的代码进行格式化 <h1>, <p>, and <li>.

h1 {
font-size: 60px;
text-align: center;
}
p, li {
font-size: 16px;
line-height: 2;
letter-spacing: 1px; }

现在让我们格式化边框。如果在 CSS 中格式不正确，我们就不能拥有文本边框。复制 下面的代码来格式化文本边框。

.borderexample {
border-style:solid;
border-color:#6d6969;
} body {
width: 900px;
margin: 0 auto;
background-color: #ffffff;
padding: 0 20px 20px 20px;
border: 5px solid rgb(255, 255, 255);
}

现在让我们做一些简单的 HTML 格式化。首先让我们做一个基本的颜色红色。

p {
  color: red;
}
