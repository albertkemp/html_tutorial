开始使用互联网

互联网是一个复杂的地方。准备工作的第一步是准备好您需要的应​​用程序和网站。

制作网站有很多步骤。但是因为总是有很多图片和视频，脚本， 样式和链接 有很多工作要使其井井有条。 但在我们继续之前，我们必须弄清楚我们如何在网络上获取它以及如何设置它。真的吗 是不是很难。首先我们需要考虑我们将如何 无需付费即可使用，GitHub 是一个绝佳的选择。在 GitHub 中设置您的帐户，您应该 准备把东西放到网上。

现在您必须准备好实际的网站。有许多不同的文本编辑器可以进行编码和制作网站 （下面有几个应用可以看）

适合初学者的优秀文本编辑器：

  • 视觉工作室代码
  • 括号
  • 原子
  • 崇高的文字
  • 记事本++

我推荐的文本编辑器是 Visual Studio Code——你将能够遵循 这些说明带有 Visual Studio 代码，因为我自己使用它。

在您下载 vscode 并在 GitHub 中设置一个帐户后，我们就可以开始制作网站了。 首先，我们必须准备好文件夹。如果你打开 vscode 它应该有文字说“编辑 更进一步”或类似的东西，但你想要的是 左上角说创建新文件。如果您没有类似的东西，请转到左上角的“文件”菜单 你的屏幕。选择第一个选项，新建文件。点击它，你面前应该有一个空白页面。 创建另一个名为styles.css 的文件并重复该文件的说明。该文件将成为您的样式表 和你的 css 所以它非常重要。 到目前为止非常好。您可能已经注意到，“新文件”顶部有一些带有链接的文本。点击 有人说编程语言之类的东西。 如果您想使用这些说明，请在搜索栏中搜索 HTML，这应该是第一个结果。如果它是 不要向下滚动一点。
