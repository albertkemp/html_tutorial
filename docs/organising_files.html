Organising Files

Now first we have to rename the file to index.html. I know that sounds like a kinda quirky name but you'll get used to it and we don't want it being called Untitled File.

Now save it as index.html and make sure it appears in your finder, then you have to get stuff organised by making some folders (in Finder). Create the first folder called web_development and another folder within it called first_site. In first_site create four folders named:

  • images
  • scripts
  • styles
  • videos

Then after you have got all that done move index.html into first_site and styles.css into the styles folder. Put any videos or images into the folders that match them if you need to. Now we are ready to do the coding.
