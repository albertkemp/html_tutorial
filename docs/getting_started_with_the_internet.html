Getting started with the internet

The internet is a complicated place. The first step to getting things ready is to get ready the apps and websites that you would need.

There are a lot of steps to making a website. But as there are always a lot of images and videos, scripts, styles and links there is a lot of work to keep it organised. But before we go any further we have to figure out how we're getting this on the web and how to set it up. Really it isn't that hard. First of all we need to consider how we are going to get it up without having to pay and a brilliant choice is GitHub. Set up your account in GitHub and you should be ready to put things on the web.

Now you have to get the actual website ready. There are many different text editors to do coding and make websites (there are a few apps below to look at)

Good text editors for beginners:

  • Visual Studio Code
  • Brackets
  • Atom
  • Sublime Text
  • Notepad++

The text editor I recommend is Visual Studio Code ‚Äî you will be able to follow these instructions with visual studio code because I use it myself.

After you have downloaded vscode and set up an account in GitHub we are ready to start making the website. First of all we have to get the folders ready. If you open up vscode it should have text saying "editing taken a step further" or something like that but what you want is the link on the top left saying create new file. If you don't have anything like that go to the File menu at the top left corner of your screen. Choose the first option, New File. Click on that and you should have a blank page in front of you. Make another file called styles.css and repeat the instructions for that as well. This file will be your stylesheet and your css so it's very important. Very good so far. As you might have noticed there's text at the top of your "New File" with some links. Click on the one saying something like programming language. If you want to use these instructions, search HTML in the search bar and that should be the first result. If it's not scroll down a bit.
