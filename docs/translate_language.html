2. HTML 入门

编码的第一步是基本文本。要使普通文本将您的文本包裹在<p> 元素 作为 如下图：

<p>Coding is so fun!</p>

Head

Heads 是代码的复杂部分，包括 <!DOCTYPE html>, the <html>, <head> and <body> 标签、徽标和标题。下面是一个你可以的头 复制。

Body

网页有两个部分，头部和主体。头部包括标题，在那里你得到了字体 在 来自 css 的徽标和 <head>, <body> and <html> 标签。 正文包括内容、图像和文本。它还包含许多标签，例如 <code>, <video>, <html>, <p>, <embed>, <h1>, <h2>, <h3>, <img>, <a>, <body>, <br> 还有很多。如果你这样做并不重要 例如 <VIDEO>, <Video> or <video>. 他们都做同样的事情。

以下是您可以复制的正文示例

<h1>My Webpage</h1>
<p>I am the content of this webpage.</p>
<img src="images/image.png" alt="Image" width="500">
</body>
</html>

Lists

列表很容易做，只包括 <ul> (or <ol>) and <li>. 第一步是选择 <ul> （无序列表）或 <ol> （有序列表）。你不做也无所谓 <ul> 或者 <ol>, 主要部分是 <li>. 第一步是做 <ul> 和 </ul> 会自动出现在后面。 下一个 步骤单击 ⮐ 并执行<li> 它也会自动出现。

在两者之间 <li> 做任何你的文字，链接，（我们将在稍后了解） 或者 无论你想列出什么。一旦这一切都完成了，那就是一个列表。该列表应如下所示：

<ul>
<li>1<li>
<li>2<li>
<li>3<li>
<ul>

Images

图像相对容易制作。在这里，我用 image.png 替换图像的名称。如果你想 添加 您自己的图像，按照这些说明或复制框中的文本，然后将 image.png 更改为 您的 image 如果您的文件中有图像，则图像还没有。

T第一步是插入 <img> 标记，这并不难。只要你输入 <img> 比它会起作用。

接下来在两者之间 <img 和 > 做 src 标签。

<img src>

然后在 src 之后做一个 = 并在“”中做图像/图像的名称。

<img src="images/image.png">

之后做 alt（意味着图像的实际名称是什么），然后做宽度标签，= 和 如何 宽你想要的。在 ””。 500只是平均值。

<img src="images/image.png" alt="Image" width="500">

Links

链接非常重要——它们将网络链接在一起。要建立链接，我们可以使用一些简单的步骤，那就是 好的 因为你可能需要很多。

首先将您的文本包装在 <a> 元素。

<a>Link</a>

接下来在两者之间放置一个 href 元素 <a 和 >.

<a href="">Link<a>

接下来插入您希望链接指向的链接。文本

<a href="https://chain.com/">Link<a>

Videos

了解如何在您的网站中插入视频非常有用。如果比视频更难，视频并不难 图片。 您是否正在尝试通过在您的网站上播放视频来向人们展示如何制作这样的教程 做 诸如此类的事情和其他目的。 视频可以通过一些简单的步骤完成。第一步是做 <video> 元素，然后在 在。。之间 <video 和 > 做 src 元素，= 包裹在你的视频名称 引述 标记，就像插入图像一样。

<video src="videos/video.mp4"

对于下一步而不是 alt 添加类型 ="video/mp4" 然后 controls> 在一个空格之后 关闭 </video> 标签。

<video src="videos/video.mp4" type="video/mp4" controls> </video>


Text Borders

Text borders aren't hard to make — they are just a simple border added to the normal <p> 元素。第一步是做 <p> 元素和在 在。。之间 <p 和 > 做 类=“边框示例”。它应该是这样的

Loops and autoplay.

做自动播放和循环并不难，首先做 src 标签，那么它基本上就像 <video> 和 <img> 没有 <video> 标签。要进行循环和自动播放，请复制下面的代码。

<video width="500" height="500" autoplay loop>
<source src="videos/video.mp4" type="video/mp4" />

Logo images

拥有徽标图像是个好主意，而且也不难。第一步是做 <link> 标签。

<link>

然后为下一步做 rel 并输入 "shortcut icon", href， 和 你的图片名称

<link rel="shortcut icon" href="images/logo.png">

这就是徽标完成，保存文件并重新加载它以查看它是否有效。
