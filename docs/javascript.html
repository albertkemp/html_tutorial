JavaScript

1. Printing

To print to the console in JavaScript is a good skill to learn. To print, simply run the console.log() function and type your string in between single or double quotes ('' or ""). For example:

console.log('Hello World!');

Output:
>>Hello World!

Notes:If you are familiar with our Python course, you will know that Python does not use semicolons. However, JavaScript does use semicolons at the end of most lines (but none after opening brackets).

2. Variables

Like nearly every other programming language, JavaScript uses variables. It is vital to know how to use them. To declare a variable, it is slightly more complicated than Python, since you have to use a keyword at the start such as var, let or const. const is used for a constant variable, which can not be reassigned later in the code. The var keyword is the general variable term that can be reassigned and is the oldest and most widespread variable which you will often find in code, but many JavaScript courses often use the let keyword since it can be easily reassigned and is easy and convenient to use. Although it is a good idea to try and use const when you don't need to reassign the variable because it uses less space which can be useful in large scale projects.

The way to declare a variable is similar to Python - you type the keyword and then the variable name followed by an = sign to assign the value which could be a string, number, or boolean.

Type Syntax
string 'Hello'
number 1
boolean true*

*make sure to use lowercase for booleans in JavaScript (the first letter is uppercase in Python).

Here is an example of how to declare a variable in JavaScript:

const word = 'hello';

To print a variable, simply wrap the variable name in the brackets (without quotes):

const word = 'hello';
console.log(word);

Output:
>>hello

3. Comments

Comments are not comments like in a chat group or on a post. Well, at least not in programming. Comments are used to label and organise code, as putting the symbols around them lets the compiler ignore whatever is inside the symbols. They can be useful if you want to put a bit of code out of action without deleting it; just wrap it in comments

Here's what JavaScript comments look like (line comments are for a single line of code, block comments are for multiline blocks of text/code.)

Type Syntax
Line comment //comment
Block comment /*comment*/

Notes: The syntax for JavaScript block comments are also used in CSS (the code that styles HTML)

4. If...else

If...else statements are used so if the code meets the criteria stated, the compiler will execute a certain function or line of code. For example:

const word = 'tomato';
if (word=='tomato') {
   console.log('Tomatoes are scary');
}

Output:
>>Tomatoes are scary

5. Loops

Loops are used to execute a function over and over again till a certain criteria is met (or it could be an infinite loop) There are two main types of loops in javascript, the while loop and the for loop. The way a while loop works is it says while a certain condition is true it will keep looping but as soon as the condition is not met, it will stop and execute a certain task

A while loop:

let i = 0;
while(i<10) {
   i++;
}

Output:
>>1
>>2
>>3
>>4
>>5
>>6
>>7
>>8
>>9

Notes: ++ means to increase by one so if a variable had the value of 1 and you added variable++ and then printed the value of the variable it would be 2.

The other type of loop, a for loop uses less lines of code, but appear a lot more complicated. First of all, the variable is declared within the for loop's brackets, then after a semicolon the condition for example "i < 10" and then finally after another semicolon the i++ part which will increase the value of i by one, and then inside the curly braces/brackets, the function to be run. In this case, it's printing 'Hello World!':

for (let i = 0; i < 10; i++) {
   console.log('Hello World!');
}

Output:
>>Hello World!
>>Hello World!
>>Hello World!
>>Hello World!
>>Hello World!
>>Hello World!
>>Hello World!
>>Hello World!
>>Hello World!

Notes: I know I have been using i as my variable for all the loops, but, remember, you can name the variable whatever you want, i is just convenient since it's short for integer.


6. And (&&), Or (||) and Equal to (=== or ==)

Unlike Python, in JavaScript instead of the simple and and or keywords you use "&&" and "||" and for equal to, "===" and "==" are interchangeable. For example:

const colorOne = 'pink'
const colorTwo = 'green'

if (colorOne == 'blue' && colorTwo == 'green' || colorOne=='pink') {
   console.log('I am happy.');
} else {
   console.log('I am sad.');
}

Output:
>>I am happy.

7. Functions

Functions in JavaScript are useful for running the same code multiple times without having to retype it. To create a function, first declare the function with the function keyword, then run the function by typing the name of the function followed by an opening and closing bracket. (()) here is an example of a function:

function sayHello() {
   console.log('Hello!')
}

sayHello();

Output:
>>Hello!

In short:

Type Syntax
Declaration function function() {}
Run function function()
