CSS Basics

CSS is the style formatting of html. Create a file called "style.css" and move it into the "styles" folder. To add some fonts, you first have to get the font families. The best way is Google Fonts (fonts.google.com). The homepage should have lots of font options. You can also search for fonts. When you've found a font you like, click on it and where it gives you styling options e.g. light, bold, italic, regular click the "Select [font]" button. After you've clicked that, copy the link that they've provided into your <head> tag - just under you're <title> or <link> tags. Then, go to the small "font-family: [font-family]" block and copy the one with your desired font. Then, insert it into the curly brackets of the element you want to format. For example:

h1 {  font-family:[font], [font-style]; }

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Crimson+Text:ital@0;1&family=Libre+Baskerville&family=Stick+No+Bills:wght@200&display=swap"

Then in styles.css (your stylesheet) insert the code below.

html {
font-size: 18px; /* px means "pixels": the base font size is now 18 pixels high */
font-family: "Times", sans-serif; /* this should be the rest of the output you got from Google fonts */
}

The code above changes the font size of all the text in the HTML document to 18 pixels high, and changes the font to "Times", sans-serif; which is a simple Times font

2. Basic CS syntax:

First let's format a HTML document. Copy the code below to format <h1>, <p>, and <li>.

h1 {
font-size: 60px;
text-align: center;
}
p, li {
font-size: 16px;
line-height: 2;
letter-spacing: 1px; }

Now let's make the border formatted. We can't have the text border if it's not nicely formatted in CSS. Copy the code below to format the text border.

.borderexample {
border-style:solid;
border-color:#6d6969;
} body {
width: 900px;
margin: 0 auto;
background-color: #ffffff;
padding: 0 20px 20px 20px;
border: 5px solid rgb(255, 255, 255);
}
