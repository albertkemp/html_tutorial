CSS Basics

CSS is the language that styles a HTML page. You can use it to add fonts, colors, size and more of HTML tags and elements. Firstly, we have to set up our CSS file. Create a file called "style.css" (or it can be anything as long as it ends with ".css"), and add it to your styles folder, or in the same main folder as the HTML file you want to change. Then, into the head of your HTML file, insert the following code (replace 'style.css' with the name of your CSS file (if it isn't 'style.css'))

<link rel="stylesheet" href="style.css">

Or you could add a style tag to the <head> tag of your HTML page as follows:

<!DOCTYPE html>
<html>
 <head>
  <title>Page title</title>
  <style>

   [Your CSS code]

  </style>
 </head>
 <body>
 </body>
</html>

Let's start by making a <p> tag red:

p {
   color: red;
}

Input Output
HTML:
<p>Hello World!</p>

CSS:
p {
   color:red;
}

Hello World!

Now let's add some fonts. Go to Google Fonts, and select a font that you like. Click on it, and it will give you a link to paste into your <head> tag. (if you haven't learnt about <head> and <body> tags, check out our HTML tutorial). First of all, paste the link into your head tag:

<!DOCTYPE html>
<html>
 <head>
  <title>Page title</title>

  [your link]

 </head>
 <body>

  [Body content]
 </body>
</html>

Next, paste the small bit of font specification (e.g. font-family: 'Times', serif;) which should be below the first link, into the tag of the desired HTML element, For example, if we want to change the font of a <h1> to Times, do this:

h1 {
   font-family: 'Times', serif;
}

The result should be the text with your desired font (if you added some text of that type in your HTML file).

Now let's change some font size. To change font size, you simply set the font-size value to a desired size (in px (pixels high)) For example:

h1 {
   font-size:60px;
}

↑This will set the font size to 60 pixels high.

Input Output
HTML:
<h1>Hello World!</h1>

CSS:
h1{
   font-size:60px;
}

Hello World!

Text Borders

To create borders around text, we will be introducing a new concept, class attribute. To set a new class, simply put a . before the name of the class that you want to assign. For example:

CSS:
.red{
   color:red;
}

HTML:
<h1 class="red">Hello World!</h1>

Output:

Hello World!

Now, we create a new class. Name it something to do with a border, so you can distinguish it from other classes. I'm calling mine ".borderexample". For example:

.borderexample{
   border-style:solid;
   border-color:#6d6969;
}

Input Output
HTML:
<p class="borderexample">Hello World!
</p>

CSS:
.borderexample{
   border-style:solid;
   border-color:#6d6969;
}

Hello World!
