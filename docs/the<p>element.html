The <p> element

The <p> element, (standing for paragraph) means the basic text, or paragraph. Let's do some experimenting with the <p> element. Let's do some basic text.

<p>Hello World! This is a paragraph</p>

What is the meaning of the <p> element?

Input: Output:
<p>Hello World</p> Hello World
<p>I like dark chocolate</p> I like dark chocolate

You can also put a border around the <p> element and put the <br> element within the <p> element.
