Getting started with HTML

1. So what is HTML?

HTML (Hypertext Markup Language) is a programming language often used in websites. HTML is not the most popular programming language (which goes to JavaScript or js) but is a good simple programming language for beginners. Let's begin writing some text.

2. Getting started with HTML

The first step to coding is basic text. To make normal text wrap your text in a <p> element as shown below:

<p>Coding is so fun!</p>

Head

Heads are complex parts of the code that include a <!DOCTYPE html>, the <html>, <head> and <body> tags, the logo, and the title. Below is a head you can copy.

<!DOCTYPE html>
<html>
<head>
<link rel="shortcut icon" href="images/logo.png">
<title>mywebsite</title>
<head>
<body>

Body

There are two parts in a webpage, the head, and the body. The head includes the title, where you got the fonts in css from, the logo and the <head>, <body> and <html> tags. The body includes the content, the images and the text. It also contains many tags such as <code>, <video>, <html>, <p>, <embed>, <h1>, <h2>, <h3>, <img>, <a>, <body>, <br> and many more. It doesn't matter if you do for example <VIDEO>, <Video> or <video>. They all do the same thing.

Below is an example of a body you can copy

<h1>My Webpage</h1>
<p>I am the content of this webpage.</p>
<img src="images/image.png" alt="Image" width="500">
</body>
</html>

Lists

Lists are quite easy to do, only including <ul> (or <ol>) and <li>. The first step is to choose between <ul> (unordered list) or <ol> (ordered list). It doesn't really matter if you don't do <ul> nor <ol>, the main part is <li>. The first step is to do <ul> and </ul> will automatically appear behind. Next step click ‚Æê and do <li> it will automatically appear as well.

In between the two <li> do whatever your text is, links, (which we will learn about later) or whatever you want to list. Once that is all done than that is a list. The list should look like this:

<ul>
<li>1<li>
<li>2<li>
<li>3<li>
<ul>

Images

Images are relatively easy to make. Here I am substituting the name of the image with image.png. If you want to add your own image, follow these instructions or copy the text in the boxes, then change image.png to the name of your image if you have images in your file, images yet.

The first step is inserting the <img> tag and that isn't hard. as soon as you type <img> than it will work.

Next in between the <img and the > do the src tag.

<img src>

Then after src do an = and do images/ the name of your image in "".

<img src="images/image.png">

After that do the alt (meaning what is a good actual name for the image) and after that do the width tag, = and how wide you want it to be. in "". 500 is just the average.

<img src="images/image.png" alt="Image" width="500">

Links

Links are very important ‚Äî they link the web together. To make a link we can use some simple steps and that's good because you're probably gonna need a lot of them.

First of all wrap your text in the <a> element.

<a>Link</a>

Next put a href element in between the <a and the >.

<a href="">Link<a>

Next insert the link that you want the link to lead to. The text

<a href="https://chain.com/">Link<a>

Videos

It is quite useful to know how to insert videos in your website. Videos aren't much harder if harder than images. Whether you are trying to make a tutorial like this one by having videos in your website to show people how to do things and other purposes like that. Videos can be done in some simple steps. The first step is to do the <video> element, then in between the <video and the > do the src element, = your name of the video wrapped in quotation marks, much like inserting image.

<video src="videos/video.mp4"

For the next step instead of alt add type ="video/mp4" then controls> and after a space the closing </video> tag.

<video src="videos/video.mp4" type="video/mp4" controls> </video>


Text Borders

Text borders aren't hard to make ‚Äî they are just a simple border added to the normal <p> element. The first step is to do the <p> element and in between the <p and the > do class="borderexample". It should look like this

<p class="borderexample">your text goes here.</p>

Loops and autoplay.

It is not hard to do autoplay and loops, first of all do the src tag, then it is basically like <video> and <img> without the <video> tag. To do loops and autoplay copy the code below.

<video width="500" height="500" autoplay loop>
<source src="videos/video.mp4" type="video/mp4" />

Logo images

It is a good idea to have a logo image, and its not hard either. The first step is to do the <link> tag.

<link>

Then for the next step do the rel and type "shortcut icon", href, and the name of your image

<link rel="shortcut icon" href="images/logo.png">

And that is the logo done, save the file and reload it to see if it worked.
