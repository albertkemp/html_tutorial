Getting started with the internet

To get started with the internet, the first step is downloading applications that you should need.

If you make a site, you have to have a way to get it up on the web. A great choice is Github, since it is free, but if you want a custom domain name you have to reserve that (which costs money) so I reccomend creating an account in Github which isn't hard.

Next, you need a text editor.

Good text editors for beginners:

  • Visual Studio Code
  • Sublime Text
  • Brackets
  • Notepad++*

*Notepad++ is only available on Windows

I most reccomend Visual Studio Code. It's a free, powerful, easy to use and very convenient text editor that you can download on Mac or Windows.

After you have downloaded a text editor and set up an account in GitHub we are ready to start organising our files
