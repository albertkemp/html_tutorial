Python

Python is a versatile and popular programming language usesd for many purposes. In this course, you will learn the basics of Python.

1. Printing

To print to the console you simply wrap a string (text) in quotes, brackets and the print() function as follows:

print('Hello World!')

Output:
>>Hello World!

Note: If you know JavaScript, unlike JavaScript, Python never uses semicolons. That's why there isn't one at the end of the line.

You can practice printing in Python with a python compiler like Programiz's Python Online Compiler.

2. Variables

Variables are an important part of almost all programming languages, and are used to store strings, numbers, or booleans like True or False. Numbers are declared without quotes, strings with single or double quotes ('' or ""), and True or False literally just True or False (with capitals)

Type Syntax
String variable = 'hello'
Number variable = 1
Boolean variable = True

To declare a variable, you simply type the name of the variable (with no declaration keyword like var, let, or int), and assign its value with an = symbol

3. Comments

Comments can be useful to label or organise certain parts of your code. They are simply text that is ignored by the compiler. To make a comment, simply put a # in front (for line comments (single line)) or put 3 double quotes ("""comment""") for block comments (multiline). For example

Type Syntax
Line comment #Comment
Block comment """Id aliquid omnis ut architecto voluptatum
non necessitatibus autem aut iste perferendis ut
exercitationem iste.
Est quod incidunt id itaque quibusdam ad
tempora ratione a modi enim sed impedit
omnis sed quas neque et veniam voluptas!
"""

4. If...else statements

If...else statements are once again used in almost all programming languages and consist of checking if a value or something else is true, then execute a command. For example

word = 'potato'
if word=='potato':
   print('The word is "potato"')
Output:
>>potato

5. Functions

Functions execute certain commands whenever they are called, so you could instead of having "print('potato')" in the if...else statement, you could replace it with "printPotato()" which would run a function that would print potato. To create a function you first define the function with the def keyword and then to run the function you type its name with an opening and closing bracket. For example:

def writePotato():
   print('potato')

word = 'potato'
if word=='potato':
   writePotato()

Functions can be useful since you only have to define what you want it to do once and then you can run the function as many times as you want.

6. Loops

Loops are used to run the same function over and over again until the criteria has been met, or it could be infinite.

Looping through an array/table:

To loop through an array (or table) you first have to define the array by creating a variable, and listing the items in the array within square brackets ([]) like this:

array = ['apple', 'orange', 'banana']

You then have to define the loop for example:

array = ['apple', 'orange', 'banana']
for x in array:
   print(x)

Output:
>>apple
>>orange
>>banana

range() function

You can use the range() function for doing the same thing but with numbers. For example:

for x in range(6):
   print(x)

Output:
>>0
>>1
>>2
>>3
>>4
>>5

the value of x changes as it loops through from 0 to 6, and if the number is <(lesser) than 6 it will print it. It basically prints the value of x until x isn't in the range from 0 to 6
