Getting started with HTML

1. So what is HTML?

HTML (Hypertext Markup Language) is a programming language used in websites. It is good to learn how to use HTML code because it is used for the framework of sites.

2. Getting started with HTML

The first step to coding is basic text. To make normal text wrap your text in a <p> element as shown below:

<p>Hello World!</p>

The result should be regular text when you load your webpage

Head

The head is a crucial element of a webpage that involves the <title> tag, and optional links to stylesheets (CSS files), shortcut icons (the little icon on the tab in your web browser), and fonts for the styles.

Let's start with simple <head> and <title> tags.

Input Output
<head>
     <title>Page title</title>
</head>

Body

There are two parts in a webpage, the head, and the body. The head includes the title, where you got the fonts in css from, the shortcut icon, and links to stylesheets or scripts.

The body includes the content, the images and the text. It also contains many tags such as <code>, <video>, <html>, <p>, <embed>, <h1>, <h2>, <h3>, <img>, <a>, <body>, <br> and many more.

Below is an example of a body.

<h1>My Webpage</h1>
<p>I am the content of this webpage.</p>
<img src="images/image.png" alt="Image" width="500">
</body>
</html>

Lists

Lists are quite easy to create, only including <ul> (or <ol>) and <li>. The first step is to choose between <ul> (unordered list) or <ol> (ordered list). Unordered list adds bullet points, so the list items aren't ranked, while ordered list (<ol>) orders it (1., 2., 3.). Decide between which one you want, and then add the <li> tag inside the <ul> or <ol> tags. Your code should look something like this:

<ul>
   <li>list item 1</li>
   <li>list item 2</li>
   <li>list item 3</li>
</ul>

   <li>list item 3</li>
</ul>

Images

Images are relatively easy to make. Here I am substituting the name of the image with image.png. If you want to add your own image, follow these instructions or copy the text in the boxes, then change image.png to the name of your image if you have images in your "images" folder yet. If you did not set up a folder check out Organising Files

The first step is inserting the <img> tag and that isn't hard. as soon as you type <img> than it will work. (Remember, the <img> tag does NOT have a closing tag - it is ONLY "<img>")

Next, in between the <img and the >, add the src attribute.

<img src>

Then after the src attribute add an = and add "images/image.png" in quotes.

Your code should resemble this:

<img src="images/image.png">

After that add the alt attribute (the alt text for people who have trouble seeing what the image is) and after that add the width attribute (which controls the viewable width of the image), = and how wide you want it to be. in "". 500 is just the average.

<img src="images/image.png" alt="Image" width="500">

Now you should have a working image in your file! (if the path to the image is all correct. Make sure you organize your files properly)

Links

Links are important. In short, they link the web together. Making a link is very simple which is good because you're probably gonna need a lot of them.

First of all wrap your text in the <a> element. (with a closing tag)

<a>Link</a>

Next put a href attribute in between the <a and the >.

<a href="">Link<a>

Next insert the link that you want the link to lead to. The text that you want the link to display should go inside the tag:

<a href="https://albertkemp.github.io/html_tutorial/">https://albertkemp.github.io/html_tutorial/<a>

Now you should have a link up and running.

Videos

It is quite useful to know how to insert videos in your website. Videos aren't much harder if harder than images. Whether you are trying to make a tutorial like this one by having videos in your website to show people how to do things and other purposes like that. Videos can be done in some simple steps. The first step is to add the <video> element, then in between the <video and the > add the src attribute, = the file name of the video wrapped in quotation marks, much like the syntax for the image.

<video src="videos/video.mp4"

For the next step add the type attribute - type="video/mp4" (if your video is a .mp4 file format. If otherwise, do otherwise.) then controls> (optionally) which will allow the user to access the video controls such as play/pause, volume, and picture-in-picture) and after a space the closing </video> tag.

Your code should look somewhat like this:

<video src="videos/video.mp4" type="video/mp4" controls > </video>

Notes: To autoplay or loop the video, simply add the autoplay or loop attributes (without any filenames, = or quotes), like so:

<video src="videos/video.mp4" type="video/mp4" controls autoplay loop></video>

You can also use the muted attribute to mute the video (<video src="videos/video.mp4" type="video/mp4" controls muted></video>)

Single quotes ('') and double quotes ("") are interchangeable


Shortcut icons

It is a good idea to have a shortcut icon (the image next to the page title in the tab, usually the site's logo), and its not hard either. The first step is to add the <link> tag.

<link>

Then for the next step add the rel attribute and type "shortcut icon". Finally, add the href attribute with the contents of the ="" being the the filepath (and filename) of the image you want to use (e.g. "./images/image.png").

<link rel="shortcut icon" href="images/logo.png">

And that's the code for a shortcut icon. Save and open the .html file and see if it worked!
